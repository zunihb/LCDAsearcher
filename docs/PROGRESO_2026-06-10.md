## Fixes Aplicados

| Problema | Fix |
|----------|-----|
| `get_trending_topics` fallaba (`_slope` no importado) | Importado desde `src.trends` |
| `search_papers` no filtraba por año | Añadidos `year_from`/`year_to` |
| `get_topic_evidence` no encontraba con acentos | Matching con forma original + normalizada |
| `get_researchers_by_topic` igual | Matching dual |
| `compare_researchers` eliminaba keywords sin match | Ahora muestra todas y marca ★ las que matchean |
| `get_trending_topics` duplicaba variantes | GROUP BY con `LOWER(keyword_norm)` |
| Chat no mantenía contexto de años | System prompt con reglas de contexto |
| Gemini no soportaba arrays en tools | Conversión recursiva de schemas |
